\section{Discussion}

The empirical behavior supports a practical interpretation of the hybrid policy. Small finite-model search is the dominant falsification mechanism: all four negative cases are refuted with order-$2$ witnesses (Lemma~\ref{lem:order2}), consistent with model-finding intuition from automated reasoning~\cite{mccune2003mace4}. This makes the countermodel branch a strong first filter for Stage-1 implication queries.

Bounded derivation improves interpretability on positive cases by producing symbolic evidence in five of six true labels. The remaining true case, $x=y\Rightarrow x\ast x=y\ast y$, is decided by fallback instead of explicit derivation. This identifies a concrete gap in the current derivation closure and motivates lightweight congruence-complete upgrades.

The broader context is complementary to recent literature. High-capacity predictive systems for equational implication~\cite{piepenbrock2021learning,axelrod2026twitch} and large ETP-scale analyses~\cite{bolan2025etp,berlioz2026latent} target coverage and representation power. Our objective is narrower: a transparent, reproducible, strict-boolean Stage-1 baseline under tight runtime and implementation constraints.

The main limitations are:
\begin{enumerate}[leftmargin=*]
\item small benchmark size (ten cases), yielding wide confidence bounds despite perfect observed accuracy;
\item finite search bound \texttt{max\_order}=3, which cannot prove implication when no witness is found;
\item intentionally incomplete derivation module.
\end{enumerate}

These limitations yield natural open questions.
\begin{conjecture}
A substantial fraction of ETP-style non-implications admit witnesses of order at most $3$.
\end{conjecture}

\begin{conjecture}
A minimal congruence-complete derivation extension can remove most fallback-true cases without materially increasing latency.
\end{conjecture}

A practical extension is to trigger ATP backends only on the fallback branch, preserving fast witness-first behavior while strengthening positive-certification coverage.
