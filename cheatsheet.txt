EQUATIONAL THEORIES CHEATSHEET (MAGMAS)
========================================

[CORE CONCEPTS]
- Magma: Set with binary op (*).
- Trivial (T): x=y. Implies all. Eq: x=y*y, x=y*z, x*x=y.
- Constant (C): x*y=z*w. All products equal a constant. Implies f(*)=g(*) if both sides are products.
- Absorption: x=x*y (Left), x=y*x (Right).
- Idempotency: x=x*x.
- Duality: L=>R iff L* => R* (swap arg order in all *).

[STRATEGY]
1. Check if L is T or C. If yes, check if R follows (T=>all, C=>products).
2. Variable Check: If R has var not in L, L must be strong (T, C, or Absorption).
   E.g., x=x*y implies x=x*z (True), but x=x*x does not imply x=x*y (False).
3. Countermodels: Small magmas (size 2 or 3) refute most false implications.
   - Constant Magma: x*y = c.
   - Left/Right Projection: x*y = x or x*y = y.
